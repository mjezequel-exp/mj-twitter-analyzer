# Expertime Template - Vue.js 3

This repository contains the sources to initiate a Vue.js 3 SPA application.

It contains the base folder structure for our SPA apps, fully configured with

-   A Vue.js 3 with TypeScript
-   Linting and formatting with ESLint and Prettier
-   Entra ID authentication with MSAL
-   A state store with Pinia
-   An HTTP client with Axios
-   Easy monitoring with Application Insights integration

## How to use

1. Clone this repository in your project  
   `git clone --depth 1 https://expertime@dev.azure.com/expertime/Expertime%20-%20Apps%20-%20Templates/_git/template-vue3 src/frontend`
2. Delete the `.git` folder from the template:  
   `rm -rf src/frontend/.git`
3. Remove this `TEMPLATE.md` file
